# 运动计划管理 App (Exercise Plan)

这是一个基于 Uni-app 开发的本地化运动计划管理工具，旨在帮助健身爱好者更高效地管理自己的训练计划、记录运动数据并追踪身体变化。

## 🌟 项目亮点

- **免登录体验**: 打开即用，无需注册账号，所有数据本地存储，保护用户隐私。
- **本地化存储**: 采用手机本地 SQLite 数据库，离线可用，响应极速。
- **灵活计划**: 支持“三分化”、“五分化”以及“自由训练”模式，自定义休息日或使用动作模板。
- **智能备械**: 根据个人器械库自动计算哑铃/杠铃配重方案，告别繁琐计算。
- **专注训练**: 沉浸式训练模式，实时记录时长、组间休息与动作进度。
- **可视化统计**: 自动生成训练量统计图表，直观展现训练分布。
- **身体追踪**: 集成摄入管理与围度追踪，全方位辅助健身体验。

## 🛠 技术栈

- **开发框架**: [Uni-app](https://uniapp.dcloud.net.cn/) (Vue 3)
- **状态管理**: [Pinia](https://pinia.vuejs.org/)
- **数据库**: SQLite (通过 HTML5+ Plus 接口实现)
- **图表库**: [uCharts](https://www.ucharts.com/)
- **UI 组件库**: [uni-ui](https://uniapp.dcloud.net.cn/component/uniui/uni-ui.html)
- **样式**: SCSS

## 📖 功能说明.

### 1. 今日训练 (首页)
- **任务概览**: 实时展示今日待练部位及动作清单。
- **励志语录**: 每日更新健身金句，激发训练动力。
- **快捷备械**: 结合个人器械库，一键获取当前动作的配重建议。
- **模式适配**: 
  - **分化模式**: 显示休息按钮，支持计划顺延。
  - **自由训练**: 隐藏休息按钮，直接加载模板动作。
- **灵活顺延**: 休息日或突发状况可一键将后续计划整体向后顺延。

### 2. 专注训练模式
- **沉浸体验**: 进入专属训练界面，自动开始训练计时。
- **实时记录**: 逐组记录重量与次数，支持递增/递减组实时调整。
- **进度追踪**: 实时显示总体进度百分比及动作完成状态。
- **内置计时**: 动作间无缝集成组间休息计时器。

### 3. 器械库管理 (智能备械)
- **个人资产录入**: 记录拥有的配重片（2.5kg, 5kg等）、哑铃杆重量、杠铃杆类型。
- **智能计算方案**: 
  - 输入目标重量，系统自动计算最优挂片方案。
  - 支持“哑铃模式”与“杠铃模式”切换。
  - 自动扣除杆重，精确匹配现有器材。

### 4. 计划管理
- **分化模式**: 支持“三分化”、“五分化”，自定义训练循环周期（如练3休1，或固定五分化），并自动在日历中生成循环课表。
- **自由训练**: 提供“自由训练模板”，由用户自由编排常用动作。保存后不生成日历计划，点击首页打卡直接显示模板动作。
- **动作编排**: 为训练日或模板设置目标部位，并从库中挑选动作、预设目标组数与次数。

### 5. 训练统计
- **维度选择**: 支持自定义时间范围查询。
- **训练量分析**: 自动计算各部位（胸、背、腿等）的训练总容量（重量 × 次数）。
- **占比分布**: 通过饼图直观展示训练重心的分布情况。

### 6. 个人中心 (身体追踪)
- **摄入管理**: 
  - 设定 TDEE (每日总能量消耗)。
  - 区分“增肌期”与“减脂期”目标。
  - 自定义每日热量及碳水、蛋白、脂肪三大营养素比例。
- **身体数据**: 
  - 记录体重、胸围、腰围、大腿围、臂围等关键数据。
  - **趋势分析**: 自动对比上一条记录，展示数据波动（上升/下降）。
- **器械管理入口**: 快速前往器械库更新自有器材。

### 7. 动作库管理
- **个性化录入**: 自由添加动作名称、分类（胸部、背部、有氧等）。
- **快捷检索**: 分类展示，方便在编排计划或打卡时快速查找。

### 8. 组间休息计时器
- **快捷计时**: 首页或训练中一键开启计时器。
- **自定义时间**: 支持手动输入时间、`+/-` 5秒微调及常用时间（30s, 45s, 60s等）快捷预设。
- **多重提醒**: 计时结束自动播放闹钟音效并触发手机震动。

### 9. 数据管理
- **备份导出**: 将所有数据（动作、计划、记录、身体数据、器械配置）导出为本地 JSON 文件。
- **还原导入**: 从备份文件一键恢复，确保更换设备或重装系统后数据不丢失。

## 🚀 快速开始

### 开发环境准备
1. 下载并安装 [HBuilderX](https://www.dcloud.io/hbuilderx.html)。
2. 克隆本项目到本地。
3. 在 HBuilderX 中导入项目。

### 运行项目
1. 在 HBuilderX 中点击 `运行` -> `运行到内置浏览器` 或 `运行到真机/模拟器`。
2. **注意**: 由于使用了 SQLite 本地数据库，大部分核心功能（如数据存储、统计等）需要在 App 环境（真机或模拟器）下运行才能获得最佳体验。

