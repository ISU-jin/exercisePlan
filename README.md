# 运动计划管理 App (Exercise Plan)

这是一个基于 Uni-app 开发的本地化运动计划管理工具，旨在帮助健身爱好者更高效地管理自己的训练计划和记录。

## 🌟 项目亮点

- **免登录体验**: 打开即用，无需注册账号，保护用户隐私。
- **本地化存储**: 所有数据均存储在手机本地 SQLite 数据库中，离线可用，响应迅速。
- **灵活计划**: 支持“三分化”、“五分化”等多种训练模式，自定义休息日和动作编排。
- **动态日历**: 根据开始日期自动生成训练课表，支持一键顺延功能，灵活应对生活突发状况。

## 🛠 技术栈

- **开发框架**: [Uni-app](https://uniapp.dcloud.net.cn/) (Vue 3)
- **状态管理**: [Pinia](https://pinia.vuejs.org/)
- **数据库**: SQLite (通过 HTML5+ Plus 接口实现)
- **UI 组件库**: [uni-ui](https://uniapp.dcloud.net.cn/component/uniui/uni-ui.html)
- **样式**: SCSS

## 📖 功能说明

### 1. 动作库管理
- 自行录入训练动作（如：杠铃卧推、深蹲等）。
- 建立个性化的动作库，方便后续编排计划。

### 2. 计划管理
- **分化设置**: 选择适合自己的训练分化（三分化/五分化）。
- **休息设置**: 设置训练循环中的休息天数（如练3休1）。
- **动作编排**: 为每一天设置目标部位及具体的训练动作、预设组数和次数。
- **开始日期**: 设置计划的正式执行日期。

### 3. 计划日历
- **自动排课**: 根据计划设置自动生成日历视图。
- **打卡记录**: 记录每日实际完成的重量（KG）、组数和次数。
- **顺延功能**: 点击“休息”可将今日及后续计划整体向后顺延一天。

### 4. 运动记录
- 按日期倒序展示所有历史运动数据。
- 支持日期查询，快速回顾往期表现。

### 5. 数据管理
- **备份导出**: 将所有动作、计划和记录导出为本地文件。
- **还原导入**: 从备份文件恢复数据，确保数据迁移或重装不丢失。

## 🚀 快速开始

### 开发环境准备
1. 下载并安装 [HBuilderX](https://www.dcloud.io/hbuilderx.html)。
2. 克隆本项目到本地。
3. 在 HBuilderX 中导入项目。

### 运行项目
1. 在 HBuilderX 中点击 `运行` -> `运行到内置浏览器` 或 `运行到真机/模拟器`。
2. **注意**: 由于使用了 SQLite 本地数据库，部分功能（如数据存储）需要在 App 环境（真机或模拟器）下运行才能获得最佳体验。
